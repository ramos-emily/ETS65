exemplo para paleta de cores:

em Fill > + > clique nos quatro pontinhos ... > Create Style.

Nomeie como:

Primary / 500 – cor principal

Primary / 700 – versão escura

Secondary / 500 – cor secundária

Neutral / 100 – fundo

Neutral / 900 – texto principal




exemplo de componente organizado: 

Criar uma frame chamada Buttons e dentro dela criar diferentes botões:

Primary / Default

Primary / Hover

Secondary / Default

Disabled




Criar estilos de texto (Text Styles):

Heading / H1

Heading / H2

Body / Regular

Body / Small


Criar outros componentes reutilizáveis:

Inputs (Input / Default, Input / Error)

Cards

Navbars

Avatares


Usar o recurso de “Auto Layout” e “Variants” nos botões, inputs, etc.
Isso ajuda a organizar e reutilizar os componentes de forma inteligente.




DICAS:


Use nomes claros e hierárquicos: Button / Primary / Large.

Marque os componentes como Main Component para serem reutilizados.

Crie grids nas páginas de layout (ex: 12 colunas para desktop).

Use Assets > Local components para acessar seus componentes facilmente
